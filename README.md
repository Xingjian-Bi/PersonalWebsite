# Personal Website

A static personal website for showcasing an introduction, experience, projects, and contact information.

## Project Structure

- `index.html`: Main website page
- `styles.css`: Page styles
- `script.js`: Interactive behavior
- `assets/`: Static assets such as icons, images, and resume files

## Local Preview

You can open `index.html` directly in a browser to preview the site.

If you prefer using a local server, run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
